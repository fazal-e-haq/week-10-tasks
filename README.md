# 📱 Flutter Week 10

This project covers responsive design, complex layouts, theming, and custom painting in Flutter.

---

## ✅ Task 1: Responsive Layouts
- Used `MediaQuery` and `LayoutBuilder`
- Breakpoints:
  - Mobile < 600
  - Tablet < 1200
  - Desktop ≥ 1200
- Created:
  - Mobile (1 column)
  - Tablet (2 columns)
  - Desktop (3 columns)
- Responsive `GridView`

---

## ✅ Task 2: Complex Layouts
- Instagram-like UI
- Used `Stack`, `Positioned`, `AspectRatio`
- Post cards with image, buttons, comments
- Stories section with horizontal scroll
- `PageView` for story viewer

---

## ✅ Task 3: Themes & Dark Mode
- Light & Dark themes using `ThemeData`
- `ChangeNotifier` for state
- Theme switch toggle
- Saved using `SharedPreferences`

---

## ✅ Task 4: Custom Painter
- Progress circle (`drawCircle`, `drawArc`)
- Signature pad (`drawPath`)
- Bar chart (`drawRect`, `TextPainter`)
- Animated clock using `AnimationController`

---

Responsive UI • CustomPainter • Animation • Theming • Layouts
